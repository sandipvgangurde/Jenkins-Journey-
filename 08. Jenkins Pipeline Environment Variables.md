## ‚ö° **Tips**

1. Use `env.VARIABLE_NAME` to access any Jenkins env variable.
2. Print all available variables with:

```groovy
sh 'printenv'
```

3. For **multibranch pipelines**, use `BRANCH_NAME`, `CHANGE_ID`, `CHANGE_AUTHOR`.
4. For build tracking, use `BUILD_NUMBER`, `BUILD_ID`, `BUILD_URL`.
5. `WORKSPACE` is key for scripts, artifacts, or file operations.
